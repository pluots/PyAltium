PyAltium Documentation
======================

PyAltium is a tool designed to be able to parse Altium files. Currently support
is only for binary schematic libraries (.SchLib).

**Quick links:** `GitHub source <https://github.com/tgross35/pyaltium/issues>`__ -
`Download/Install <https://pyaltium.readthedocs.io/en/latest/installation.html>`__ -
`Documentation <https://pyaltium.readthedocs.io/en/latest/>`__ -
